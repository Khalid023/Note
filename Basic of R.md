# Basic of R
## Data Types
### Objects
five basic objects  
- character
- numeric(实数)
- integer
- complex
- logical(True/Flse)

**向量vector 是最基本的对象但只能包含一种形式的对象。使用vector()创建**  
**列表list 可以包含多种对象**

### Attributes

R objects can have attributes  
- names，dimanames
- dimensions
- class
- length
- metadata
- etc

**使用attributes()创建属性**
